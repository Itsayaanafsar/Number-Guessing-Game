## Description
This is a simple Number Guessing Game built using Python.  
The program generates a random number within a user-defined range, and the player has a limited number of attempts to guess the correct number.

This project was created to practice Python basics such as loops, conditionals, exception handling, and the random module.

---

## Features
- User-defined number range
- Random number generation
- Limited number of attempts
- Input validation using try-except
- Feedback for each guess (Too high / Too low)
- Best score is tracked during the current game session.

- Python
- random module
